# mfaldosari.github.io

Personal portfolio for Mohammad Fahad Aldosari — DevOps & Cloud Infrastructure Engineer (OpenShift · Kubernetes · Data Engineering).

Bilingual (English / Arabic, full RTL support), single-page, dark theme. Plain HTML/CSS/JS — no build step, deploys directly via GitHub Pages.

## Structure

- `index.html` — page content (all sections, both languages)
- `css/style.css` — styling, theme, responsive layout
- `js/main.js` — language toggle, mobile nav, scroll reveal
- `assets/` — favicon

## Local preview

```
python3 -m http.server 8080
```

Then open http://localhost:8080
